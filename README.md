# Homebrew Tap for Kuark System

Multi-agent AI development system for Claude Code.

## Installation

```bash
brew tap kuarkdijital/kuark
brew install kuark-system
```

## Update

```bash
brew upgrade kuark-system
```

## Uninstall

```bash
brew uninstall kuark-system
```

## What is Kuark System?

Kuark is a multi-agent development system that extends Claude Code with 16 specialized AI agents. These agents collaborate through a swarm protocol to build full-stack applications using NestJS, Next.js, Prisma, and more.

### Agents (16)

| Agent | Role |
|-------|------|
| Product Owner | Vision, backlog, requirements |
| Project Manager | Sprint planning, task distribution |
| Analyst | Requirements analysis, user stories |
| Architect | Architecture decisions, ADR |
| NestJS Developer | Backend API development |
| NextJS Developer | Frontend development |
| UI/UX Designer | Wireframes, design system, component specs |
| Database Engineer | PostgreSQL, Prisma schema |
| Queue Developer | BullMQ, background jobs |
| QA Engineer | Testing, coverage |
| Security Engineer | Auth, RBAC, vulnerability |
| DevOps Engineer | Docker, Railway, CI/CD |
| API Researcher | 3rd party API integration |
| Documentation | Technical docs, API docs |
| Python Developer | FastAPI microservices |
| Orchestrator | Agent coordination |

### Usage

Start Claude Code in any project directory and say **"proje baslat"** to begin.
